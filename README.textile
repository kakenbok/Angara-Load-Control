h3. Features

* Supports sequences and group of sequences
* Stop and resume loading for a single item, a sequence or a group of sequences
* Loader callbacks with stopOnCallback
* Monitoring of individual items, sequences, loading at all
* Paged sequences (loading only of visible parts of a sequence)
